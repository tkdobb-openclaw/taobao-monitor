# 小红书运营技能

专业的小红书内容创作和运营支持。

## 核心功能

### 1. 选题策划
- 热点话题追踪
- 竞品内容分析
- 用户痛点挖掘
- 系列内容规划

### 2. 文案创作
- 笔记正文撰写
- 标题优化
- 标签生成
- 封面文案建议

### 3. 发布策略
- 最佳发布时间
- 发布频率建议
- 内容排期表

## 使用方式

### 生成完整笔记
```bash
./scripts/generate-note.sh --topic "潜水OW考证避坑" --type "攻略"
```

### 标题优化
```bash
./scripts/optimize-title.sh "原标题"
```

### 生成标签
```bash
./scripts/generate-tags.sh "潜水 考证 泰国"
```

## 内容模板

### 攻略类笔记结构
1. **痛点引入** (1-2句) - 击中目标用户痛点
2. **个人经历** (3-5句) - 建立信任感
3. **干货内容** (分点列出) - 核心价值
4. **避坑提醒** (3-5点) - 实用性强
5. **CTA引导** (1句) - 互动或关注

### 种草类笔记结构
1. **场景化开头** - 营造使用场景
2. **产品亮点** (3-5个) - 突出卖点
3. **真实体验** - 增加可信度
4. **价格/购买渠道** - 降低决策门槛
5. **总结推荐** - 强化印象

## 标题公式

参考 [references/title-patterns.md](references/title-patterns.md) 获取完整标题模板。

常用公式：
- 数字法：「3个技巧」「5个避坑点」
- 对比法：「别去XX，去这里」「新手vs老手」
- 悬念法：「后悔没早知道」「我发现了...」
- 身份法：「打工人必看」「宝妈必备」

## 标签策略

- 精准标签 (2-3个)：细分领域词
- 热门标签 (2-3个)：大流量词
- 长尾标签 (2-3个)：场景/人群词
- 品牌标签 (1个)：个人IP词

## 最佳发布时间
