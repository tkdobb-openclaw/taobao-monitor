### 🚨 重要教训

**GitHub Pages 分支问题：**
- `taobao-monitor` 仓库的 GitHub Pages 部署的是 **master** 分支
- 不是默认的 main 分支！
- 以后推送必须同步到 master：`git checkout master && git merge main && git push`
- 否则网页显示的是旧数据
