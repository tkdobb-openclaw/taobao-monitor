# 淘宝价格监控清单模板

## 填写说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| **商品名称** | ✅ | 商品型号/名称，用于识别 | Shearwater Peregrine TX |
| **商品链接** | ✅ | 淘宝/天猫商品详情页链接 | https://item.taobao.com/item.htm?id=624281587175 |
| **店铺名称** | ✅ | 店铺名，用于区分不同店 | 大洋潜水中心 |
| **当前价格** | ❌ | 当前售价（可不填，系统会自动抓取） | 3360 |
| **目标价格** | ❌ | 低于此价格时提醒（可不填） | 3000 |
| **备注** | ❌ | 其他信息 | 官方授权店/代理商 |

## 示例数据

```csv
商品名称,商品链接,店铺名称,当前价格,目标价格,备注
Shearwater Peregrine TX,https://item.taobao.com/item.htm?id=624281587175,大洋潜水中心,3360,3000,官方授权店
Shearwater Peregrine TX,https://item.taobao.com/item.htm?id=xxxxxxx,塞班户外,3380,3000,代理商
```

## 使用步骤

1. 复制 `监控清单模板.csv` 文件
2. 用 Excel 或 Numbers 打开
3. 按示例格式填写你的监控商品
4. 保存为 CSV 格式（UTF-8 编码）
5. 导入系统：`python3 scripts/monitor.py import 你的文件.csv`

## 注意事项

- 同一商品可添加多个店铺链接，系统会自动对比最低价
- 商品链接必须是淘宝或天猫的详情页链接
- 目标价格可不填，填了会在低于此价时通知
- 当前价格可不填，系统会自动抓取
